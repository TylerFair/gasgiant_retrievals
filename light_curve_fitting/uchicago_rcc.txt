First: 
1. download miniconda3 from instructions but point to project directory
2. create a conda env with conda create -n <conda env> python=3.10
3. follow environment.txt
4. Make a something.sbatch script with the template below

SBATCH FILE: 
#!/bin/bash
#SBATCH --output=job-info.out
#SBATCH --error=job-info.err
#SBATCH --ntasks=1
#SBATCH --account=pi-<piname>
#SBATCH --partition=gpu
#SBATCH --cpus-per-task=3
#SBATCH --mem-per-cpu=30G
#SBATCH --gres gpu:1
#SBATCH --constraint=a100
source /project/<pi>/<cnetid>/miniconda3/bin/activate
conda activate /project/<pi>/<cnetid>/miniconda3/envs/<conda env>
module load cuda/12.9
python /project/<pi>/<cnetid>/fit_jwst.py -c /project/<pi>/<cnetid>/config.yaml



### Helpful rcc commands
sbatch <script>.sbatch # starts script
squeue --state=RUNNING --user=<cnetid> # checks script 
scancel <id> # to cancel script if error
